# Brief Description (for submission form)

JanSetu AI is a decision-intelligence platform that turns citizen voices into sanctioned constituency development works. India's MPLADS scheme has ₹1,729 crore lying unspent and 62% of MPs miss the mandatory 22.5% SC/ST-area allocation — existing tools collect complaints (CPGRAMS, Swachhata) or track works after sanction (e-SAKSHI), but nothing helps an MP's office decide *which* works to sanction. JanSetu fills that gap. Citizens report civic issues by photo (Gemini Vision auto-classifies category, department, severity), by voice in 12 Indian languages, or text — working fully offline via a PWA queue. The MP's office asks questions in natural language in any Indian language (Gemini NL→SQL, guard-railed and read-only), sees ward-level demand hotspots on Google Maps, gets statistical spike alerts and an SC/ST compliance radar with corrective amounts, and generates a Priority Works briefing where Gemini fuses demand, demographics, and the MPLADS fund ledger into ranked, costed, compliant sanctionable works — each with a one-click draft recommendation letter to the District Authority. Built on Gemini 2.0 Flash, BigQuery, Cloud Translation API, Cloud Run (asia-south1), and Google Maps Platform, with a three-tier graceful-degradation chain (Gemini → Groq → built-in Lite engine) that keeps every feature functional even with zero API keys. One deployment serves a ~25 lakh-citizen constituency; onboarding a new one takes under a day.
